# 🌍 T2T — Travel to Tell

> *"Il mondo è un libro, e chi non viaggia ne legge solo una pagina."*

**T2T** è un social network per viaggiatori in formato **diario di viaggio interattivo**.
Condividi video, foto, esperienze e consigli con una community globale di esploratori.

## ✨ Features (MVP)

- 📖 **Travel Diary** — Crea diari strutturati giorno per giorno con rich content
- 🗺️ **Interactive Map** — Mappa personale con tutti i luoghi visitati
- 💡 **Tips & Consigli** — Sistema strutturato di consigli su alloggi, ristoranti, trasporti
- 👥 **Social Layer** — Follow, like, commenti, condivisione itinerari
- 🔍 **Discovery** — Feed personalizzato, trending destinations, ricerca per meta
- 🎮 **Gamification** — Badge, statistiche, scratch map

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| **Mobile** | React Native + Expo |
| **Backend** | Supabase (Auth, DB, Storage, Realtime) |
| **Database** | PostgreSQL + PostGIS |
| **Maps** | Google Maps |
| **i18n** | English + Italiano |
